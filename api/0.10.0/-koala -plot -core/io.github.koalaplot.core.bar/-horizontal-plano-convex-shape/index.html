@Stable
class HorizontalPlanoConvexShape<X, E : HorizontalBarPlotEntry<Float, X>>(xyGraphScope: XYGraphScope<Float, X>, index: Int, value: E) : Shape, XYGraphScope<Float, X>

Rectangle shape with planar/convex shaped sides. Useful for Single Horizontal Bar and Stacked Bars Plot rendering.

Provides access to yAxisModel and acts as an implementation of XYGraphScope.

index

Represents the element index within the series.

value

The HorizontalBarPlotEntry that defines the cutouts for the HorizontalPlanoConvexShape.
